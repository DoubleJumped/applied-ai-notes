    - [keynote](playbooks/fde-presentation-keynote.html) — Jobs / presentation-zen: dark gradient, one idea per slide, billboard rule
    - [takahashi](playbooks/fde-presentation-takahashi.html) — Takahashi method: 40 rapid slides, giant text only
    - [sparkline](playbooks/fde-presentation-sparkline.html) — Duarte's Resonate structure: "what is" vs "what could be", with a live sparkline
  - Ten more, each implementing a documented school of presentation design:
    - [kawasaki](playbooks/fde-presentation-kawasaki.html) — Guy Kawasaki's 10/20/30 rule: 10 slides, nothing under 30pt
    - [pechakucha](playbooks/fde-presentation-pechakucha.html) — PechaKucha 20×20: 20 visual slides, 20-second auto-advance with countdown ring
    - [memo](playbooks/fde-presentation-memo.html) — the Bezos six-page narrative memo: no slides, prose read in silence
    - [tufte](playbooks/fde-presentation-tufte.html) — Tufte evidence handout: sentence headlines, margin sidenotes, sparklines, zero chartjunk
    - [swiss](playbooks/fde-presentation-swiss.html) — International Typographic Style: visible grid, Helvetica, red geometry, lowercase
    - [bauhaus](playbooks/fde-presentation-bauhaus.html) — primary colors with a meaning-bearing shape legend (square/circle/triangle)
    - [chalkboard](playbooks/fde-presentation-chalkboard.html) — Feynman-lecture blackboard: chalk text, hand-drawn diagrams, 18 boards
    - [zoom](playbooks/fde-presentation-zoom.html) — Prezi-style zooming canvas: the whole methodology as one spatial loop map
    - [editorial](playbooks/fde-presentation-editorial.html) — magazine feature: Didot display, drop caps, columns, pull quotes
    - [brutalist](playbooks/fde-presentation-brutalist.html) — web brutalism: hard borders, offset shadows, caution yellow, mono metadata
